# Анализ результатов пространственного анализа (4 типа поведения, этаж 0)

## 1. Исходные данные

- **Типы:** Активный обходчик, Быстрый, Исследователь, Медленный (всего 4 типа после объединения «Быстрый» и «Быстрый краткий»).
- **Траектории:** 45 (объединение с floor0_trajectories_with_features по trajectory_id).
- **Пространство:** координаты x, y в единицах плана этажа 0; квадранты SW, SE, NW, NE относительно медианных x, y по всем точкам.

---

## 2. Характеристики типов (поведение + пространство)

### 2.1. Активный обходчик (7 траекторий, 4981 точка)

**Поведение:** высокая скорость (32.02), средняя длительность (3349 с ≈ 56 мин), много экспонатов (188), мало остановок (51).

**Пространство:**
- Центр активности: X ≈ 4483, Y ≈ 1670.
- Разброс: σ_x ≈ 1520, σ_y ≈ 1060.
- Охват: по X ≈ 6712, по Y ≈ 4894.

**Квадранты:** SE 37.8% > NW 27.8% > SW 18.0% > NE 16.4%.  
Наиболее выраженное смещение в **юго-восток (SE)**.

**Интерпретация:** быстрые, «обходчиковые» траектории концентрируются в юго-восточной части этажа; при этом охват пространства большой — тип использует этаж широко.

---

### 2.2. Быстрый (13 траекторий, 5598 точек)

**Поведение:** высокая скорость (31.39), короткая длительность (2497 с ≈ 42 мин), меньше экспонатов (106), мало остановок (39).

**Пространство:**
- Центр активности: X ≈ 4359, Y ≈ 1828.
- Разброс: σ_x ≈ 1707, σ_y ≈ 1096.
- Охват: по X ≈ 6687, по Y ≈ 4939.

**Квадранты:** NW 35.6% > SE 27.5% > NE 19.7% > SW 17.2%.  
Преобладает **северо-запад (NW)**.

**Интерпретация:** быстрые короткие визиты чаще связаны с северо-западной частью этажа; центр смещён севернее и западнее по сравнению с Активным обходчиком.

---

### 2.3. Исследователь (14 траекторий, 8346 точек)

**Поведение:** средняя скорость (22.61), долгая длительность (3946 с ≈ 66 мин), много экспонатов (193), много остановок (73).

**Пространство:**
- Центр активности: X ≈ 4416, Y ≈ 1658.
- Разброс: σ_x ≈ 1624, σ_y ≈ 1068.
- Охват: по X ≈ 6713, по Y ≈ 5010 — **наибольший охват по Y** среди типов.

**Квадранты:** SE 39.2% > NW 31.3% > SW 16.6% > NE 12.8%.  
Явное предпочтение **юго-востока (SE)** при заметной доле NW.

**Интерпретация:** «исследовательское» поведение (долго, много остановок и экспонатов) сильнее всего проявляется в юго-восточной части этажа; тип охватывает пространство шире остальных по вертикали (Y).

---

### 2.4. Медленный (11 траекторий, 8903 точки)

**Поведение:** низкая скорость (18.84), самая долгая длительность (4062 с ≈ 68 мин), мало экспонатов (110), среднее число остановок (61).

**Пространство:**
- Центр активности: X ≈ 4255, Y ≈ 1822 — **самый смещённый на запад** (наименьший avg_x).
- Разброс: σ_x ≈ 1738 (максимальный), σ_y ≈ 935 (минимальный).
- Охват: по X ≈ 6727, по Y ≈ 4917.

**Квадранты:** NW 38.8% > SE 30.9% > NE 16.6% > SW 13.7%.  
Наибольшая доля в **северо-западе (NW)** среди всех типов.

**Интерпретация:** медленные долгие визиты с меньшим числом экспонатов сильнее привязаны к северо-западной зоне; при этом разброс по X максимальный — тип «размазан» по горизонтали, но по Y концентрированнее (минимальный σ_y).

## 3. Сравнение типов по пространству

|-------------------|----------------|--------------------------|----------------------------------------------|

### 3.1. Пары «поведение — зона»

- **Юго-восток (SE):** Исследователь и Активный обходчик — оба с высокой «продуктивностью» (много экспонатов) и либо долгим визитом, либо высокой скоростью. SE интерпретируется как зона интенсивного использования.
- **Северо-запад (NW):** Быстрый и Медленный — оба с меньшим числом просмотренных экспонатов; различие в скорости и длительности, но пространственный паттерн (предпочтение NW) сближает их. NW — зона менее плотного просмотра или более транзитного/созерцательного движения.

---

## 4. Выводы для архитектурного анализа (этаж 0)

1. **Разделение зон по типам использования:**
   - Зоны с преобладанием **SE** (юго-восток): более интенсивное, «исследовательское» и обходческое использование; целесообразно закладывать больше мест для остановок, комфортный просмотр, возможность долго находиться.
   - Зоны с преобладанием **NW** (северо-запад): больше быстрых и медленных траекторий с меньшим числом экспонатов; можно учитывать транзитность и меньшую плотность остановок.

2. **Тип «Медленный»:** максимальная привязка к NW и самый западный центр (min avg_x); при проектировании северо-западной части этажа имеет смысл учитывать сценарии долгого, но не обязательно «насыщенного экспонатами» пребывания (освещение, скамьи, ширина путей).

3. **Тип «Исследователь»:** максимальный охват по Y и выраженный SE; юго-восточная зона — кандидат на зоны с высокой концентрацией экспонатов и комфортом длительного просмотра.

4. **Ограничения:** анализ только по этажу 0; квадранты заданы по медиане координат и носят описательный характер. **Привязка к экспонатам и стенам** выполнена отдельно (скрипт `spatial_analysis_exhibits.py`, отчёт `spatial_exhibits_report_4types.md`): по данным start_obs_artworks и artworks_dataset построены таблицы по комнатам/стенам и топ экспонатов по типам, а также heatmap наблюдаемости по стенам на плане этажа 0.
