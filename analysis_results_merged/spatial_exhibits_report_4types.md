# Расширение этапа 4: привязка к экспонатам и стенам (4 типа, этаж 0)

## 1. Источники данных и метод

- **Типы поведения:** из `floor0_trajectories_clustered_merged.csv` (4 типа: Активный обходчик, Быстрый, Исследователь, Медленный).
- **Наблюдения экспонатов:** `start_obs_artworks/items_<trajectory_id>.csv` — для каждой траектории файл с полями *timestamp*, *floorNumber*, *paintingId*. Одна строка = момент времени, когда посетитель наблюдал экспонат *paintingId*.
- **Каталог экспонатов:** `artworks_dataset.csv` — по полю *image id* (= *paintingId*) получены *room_id*, *wall_id*, *painting name* для этажа 0.
- **План этажа:** `NMFA_3floors_plan.json` — геометрия стен этажа 0 (id, position — отрезки в координатах плана).

**Объединение:** по *trajectory_id* к наблюдениям добавлен тип поведения; по *paintingId* наблюдения соединены с каталогом (комната, стена, название). Учтены только наблюдения этажа 0.

**Результат объединения:** 2098 наблюдений, 45 траекторий, 135 экспонатов этажа 0 в каталоге, 235 стен на плане.

---

## 2. Результаты по комнатам и стенам

### 2.1. По комнатам (spatial_observations_by_room_type.csv)

Для каждого типа поведения посчитаны: число наблюдений (*n_observations*), число траекторий (*n_trajectories*), доля наблюдений типа в % (*pct_obs*).

**Примеры интерпретации:** у «Активный обходчик» наибольшие доли по комнатам — Room_7 (19.36%), Room_13 (14.77%), Room_14 (12.77%), Room_12 (11.58%), Room_10 (10.38%). У «Быстрый» лидирует Room_7 (24.13%), затем Room_12 (13.94%), Room_13 (12.06%). Комната Room_7 сильно используется обоими типами; различие в остальных комнатах отражает разницу паттернов обхода.

### 2.2. По стенам (spatial_observations_by_wall_type.csv)

Аналогичная агрегация по *wall_id*. Позволяет сопоставлять типы по конкретным стенам и использовать эти данные для heatmap на плане.

### 2.3. Топ экспонатов по типам (spatial_top_exhibits_by_type.csv)

Для каждого типа отобраны до 15 экспонатов с наибольшим числом наблюдений: *paintingId*, *painting name*, *room_id*, *wall_id*, *n_observations*, *n_trajectories*.

**Примеры:** у «Активный обходчик» в топе — La Toussaint (0_0861, Room_13), Les Amoureux (0_0841), La douleur (0_0869), Nymphe (0_0887), Les Voluptueux (0_0890) и др. У «Быстрый» — Les Voluptueux (0_0890), La douleur (0_0869), Pipe d'eau douce (0_0731), La Toussaint (0_0861), Les grands Trans-parents (0_5830) и др. Общие «хиты» (Les Voluptueux, La douleur, La Toussaint) при разном составе остальных — признак общих точек притяжения при различии обхода.

## 3. Визуализации на плане

- **spatial_walls_heatmap_all_types.png** — все стены этажа 0 отрисованы отрезками; цвет по суммарному числу наблюдений экспонатов на этой стене (по всем типам). Выделяются наиболее «нагруженные» стены.
- **spatial_walls_heatmap_by_type.png** — четыре карты (по одному на тип): для каждого типа свои интенсивности по стенам. Удобно сравнивать, какие зоны/стены важны для Активного обходчика, Быстрого, Исследователя, Медленного.

---

## 4. Выводы для проектирования и агентной модели

1. **Комнаты и стены:** Room_7, Room_12, Room_13, Room_10 — зоны высокой наблюдаемости у нескольких типов; при расчёте нагрузки и вместимости их имеет смысл учитывать отдельно.
2. **Топ экспонаты:** таблица *spatial_top_exhibits_by_type.csv* даёт конкретные произведения и стены для калибровки агентов (например, вероятности остановки у определённых стен или экспонатов по типам).
3. **План:** heatmap по стенам позволяют находить «мёртвые» стены (мало наблюдений) и перегруженные и увязывать их с типами поведения.
