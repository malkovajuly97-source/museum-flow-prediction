# Расшифровка и интерпретация: планировка и типы поведения

Краткое объяснение результатов проверки «есть ли отличия по признакам планировки и изовиста между типами поведения» (Kruskal–Wallis по 34 признакам, 45 траекторий, 4 типа: Активный обходчик, Быстрый, Исследователь, Медленный).

---

## Что проверяли

По каждой траектории посчитаны признаки **планировки** (ширина прохода, связность зон, глубина от входа, возвраты, близость к выходу, лестницы, повороты и т.д.) и **изовиста** (площадь и периметр видимости). Для каждого признака проверяли: **различаются ли средние значения между четырьмя типами поведения**. Если да (p < 0.05), тип можно частично описать через среду; если нет — по этому признаку типы ведут себя одинаково.

---

## Итог в двух словах

- **Значимых отличий: 4 из 34** — по большинству признаков планировки и изовиста типы **не различаются** (ширина прохода, изовист, тупики, лестницы, повороты и др.).
- **Различаются** типы по: **оставшемуся пути до выхода**, **глубине от входа**, **доле конца визита** и **доле пути с возвратами**. То есть типы по-разному связаны с «где человек находится по маршруту и по времени», а не с «насколько узко/открыто/видимо» в среднем.

---

## Расшифровка четырёх значимых признаков

### 1. H10_mean_path_remaining — оставшийся путь до выхода

**Что это:** В среднем по точкам траектории: сколько ещё пути остаётся до выхода (по длине маршрута). Чем больше значение, тем дольше человек в среднем «далеко от выхода».

**По типам (среднее):**
- **Активный обходчик** — ~26 615 (самый высокий)
- **Медленный** — ~22 929
- **Исследователь** — ~17 851
- **Быстрый** — ~16 952 (самый низкий)

**Интерпретация:** У **Быстрого** и **Исследователя** маршруты в среднем чаще проходят ближе к выходу (меньше «оставшийся путь»). У **Активного обходчика** и **Медленного** — больше путь вдали от выхода. То есть типы по-разному используют пространство относительно выхода: одни раньше оказываются near exit, другие дольше остаются «в глубине».

**Для модели:** Имеет смысл задавать по типам разную типичную величину «оставшегося пути» или разную склонность держаться ближе/дальше от выхода (например, для Быстрого — чаще ближе к выходу, для Активного обходчика — больше путь в глубине).

---

### 2. H4_mean_depth — глубина от входа

**Что это:** Доля пройденного пути от начала визита (0 = только начало, 1 = конец). Среднее по точкам — насколько «глубоко» в среднем проходит маршрут.

**По типам (среднее):**
- **Исследователь** — ~0,49
- **Быстрый** — ~0,48
- **Медленный** — ~0,43
- **Активный обходчик** — ~0,39 (самый низкий)

**Интерпретация:** У **Исследователя** и **Быстрого** маршруты в среднем «глубже» (больше середина и конец пути). У **Активного обходчика** и **Медленного** — больше доля пути в «начальной» половине визита. То есть типы по-разному распределяют путь по фазе визита (начало vs середина/конец).

**Для модели:** Можно задавать по типам разную типичную глубину маршрута (например, Исследователь/Быстрый — глубже, Активный обходчик/Медленный — меньше глубина в среднем).

---

### 3. H7_pct_end — доля пути в конце визита

**Что это:** Доля точек траектории, попадающих в последнюю треть времени визита («конец визита»).

**По типам (среднее):**
- **Быстрый** — ~0,46 (почти половина пути в конце)
- **Исследователь** — ~0,41
- **Медленный** — ~0,11
- **Активный обходчик** — ~0,10

**Интерпретация:** **Быстрый** и **Исследователь** проводят большую долю визита в **конце** (последняя треть времени). **Активный обходчик** и **Медленный** — наоборот, большую долю в **начале** (у них высокие H7_pct_start). То есть типы различаются по временной структуре: «много в конце» vs «много в начале».

**Для модели:** Имеет смысл задавать по типам разное распределение по фазам визита (доля начала/середины/конца), например у Быстрого/Исследователя — больше вес конца, у Активного обходчика/Медленного — больше вес начала.

---

### 4. H8_pct_path_revisit — доля пути с возвратами

**Что это:** Доля точек траектории, приходящихся на уже посещённые зоны (повторное заход в ту же зону).

**По типам (среднее):**
- **Медленный** — ~0,977
- **Активный обходчик** — ~0,978
- **Исследователь** — ~0,971
- **Быстрый** — ~0,966 (самый низкий)

**Интерпретация:** Все типы имеют **очень высокую** долю пути с возвратами (96–98%). Различие небольшое, но достоверное: у **Быстрого** чуть меньше доля возвратов, у **Медленного** и **Активного обходчика** — чуть больше. То есть «возвратность» маршрута слабо, но значимо связана с типом.

**Для модели:** Можно задавать по типам небольшие различия в склонности к возвратам (Быстрый — чуть меньше доля повторных зон, Медленный/Активный обходчик — чуть больше).

---

## Признаки, по которым типы не различаются

По этим признакам **нет достоверных отличий** между типами (p ≥ 0.05). То есть по ним типы ведут себя в среднем одинаково, и их можно задавать **общей моделью** (одни и те же зависимости от планировки для всех):

- **Ширина и узкие места:** passage_width_mean, H2_min_width, H2_pct_narrow, H9_width_std, H9_width_range  
- **Связность зон:** H3_mean_connectivity, H3_pct_low_connectivity  
- **Коридоры и залы:** H5_pct_corridor, H5_mean_aspect_ratio  
- **Плотность экспонатов:** H6_mean_exhibit_density  
- **Лестницы:** H11_mean_dist_to_staircase, H11_min_dist_to_staircase, H11_pct_near_staircase  
- **Повороты:** H12_n_turns, H12_turns_per_length, H12_turns_per_minute  
- **Изовист:** isovist_area_mean, isovist_area_min, isovist_area_max, isovist_area_range, isovist_perimeter_mean  
- **Открытость/размер зон:** zone_area_mean, pct_small_zone, pct_large_zone  

Вывод: влияние «узко/широко», «открыто/закрыто», «видимость», «лестницы», «повороты» на движение можно задавать **общими правилами для всех агентов**; тип при этом различается по **расположению маршрута во времени и пространстве** (путь до выхода, глубина, фаза визита, доля возвратов).

---

## Практический вывод для модели поведения

1. **Тип-специфичные настройки** задавать по четырём признакам:
   - типичный «оставшийся путь до выхода» (или склонность быть ближе/дальше от выхода);
   - типичная «глубина» маршрута от входа;
   - распределение по фазам визита (доля начала / середины / конца);
   - слабые различия по доле пути с возвратами.

2. **Общая модель** — для всех признаков планировки и изовиста из списка выше: одни и те же зависимости «узкость → замедление», «видимость → скорость» и т.д. для всех типов.

3. **Итог:** Типы различаются не столько по «в какой среде ходят», сколько по **где и когда** они находятся на маршруте (близость к выходу, глубина, фаза визита, возвраты). Среду (ширина, изовист, лестницы и т.д.) можно описывать общими правилами; тип задаёт «профиль маршрута» по времени и пространству.
