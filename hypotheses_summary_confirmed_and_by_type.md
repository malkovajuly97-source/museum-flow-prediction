# Гипотезы: что подтвердилось, что берём в работу, что по типам и что общее

Краткий свод: **какие гипотезы считаем подтверждёнными и берём в модель**, и по каким из них задавать **разные настройки по типам поведения**, а по каким — **одну общую зависимость для всех**.

---

## 1. Подтверждённые гипотезы (берём в работу)

Ниже — гипотезы, по которым связь с поведением (скорость, остановки, длительность, число экспонатов) подтверждена по данным. Все их используем в модели поведения.

| № | Гипотеза | Результат | Сила |
|---|----------|-----------|------|
| **1** | Ширина прохода | Уже проход → чаще останавливаются, идут медленнее | Сильная |
| **2** | Узкие места (бутылочное горлышко) | Больше узких точек → чуть меньше экспонатов, выше интенсивность остановок | Слабая |
| **3** | Тупики vs проходные зоны | Тупики → медленнее и меньше картин; проходные зоны → больше остановок | Сильная |
| **4** | Глубина от входа | Глубже в музей → медленнее, больше остановок | Подтверждена |
| **5** | Коридоры vs залы | Коридоры → чуть меньше просмотренных картин | Слабая |
| **6** | Плотность экспонатов по маршруту | Больше экспонатов по пути → быстрее идут и больше смотрят | Сильная |
| **7** | Начало / середина / конец визита | Начало — быстрее; конец — больше остановок | Подтверждена |
| **8** | Возвраты в залы | Больше возвратов → дольше визит, больше остановок, ниже скорость | Самая сильная |
| **9** | Смена узких и широких участков | Больше смена → медленнее визит, меньше картин | Умеренная |
| **10** | Близость к выходу | Дальше от выхода по пути → дольше визит, больше экспонатов | Подтверждена |
| **11** | Близость к лестницам | Ближе к лестнице → ниже скорость, дольше визит, больше остановок | Заметная |
| **12** | Количество поворотов | Больше поворотов → дольше визит, больше остановок; больше поворотов в минуту → выше скорость | Подтверждена |
| **Изовист** | Видимость (площадь изовиста) | Больше видимая площадь → выше скорость, меньше остановок; меньше nb_items при большей видимости | Сильная по stop_intensity |

## 2. Различаются по типам поведения (тип-специфичные настройки)

По этим четырём признакам типы поведения **достоверно различаются** (Kruskal–Wallis, p < 0.05). В модели задаём **разные параметры для каждого типа**.

| Признак | Гипотеза | Что задавать по типам |
|---------|----------|------------------------|
| **H10_mean_path_remaining** | Близость к выходу (оставшийся путь) | Быстрый и Исследователь — в среднем ближе к выходу; Активный обходчик и Медленный — больше путь вдали от выхода. Задать типичный «оставшийся путь» или склонность быть ближе/дальше от выхода по типам. |
| **H4_mean_depth** | Глубина от входа | Исследователь и Быстрый — глубже (≈0,48–0,49); Активный обходчик и Медленный — меньше глубина (≈0,39–0,43). Задать типичную глубину маршрута по типам. |
| **H7_pct_end** | Доля пути в конце визита | Быстрый и Исследователь — большая доля в конце визита (≈0,41–0,46); Активный обходчик и Медленный — большая доля в начале. Задать распределение по фазам визита (начало/середина/конец) по типам. |
| **H8_pct_path_revisit** | Доля пути с возвратами | У всех высокая (96–98%); у Быстрого чуть ниже (≈0,97), у Медленного и Активного обходчика чуть выше (≈0,98). Задать небольшие различия по склонности к возвратам по типам. |

**Итог:** Для гипотез **4, 7, 8, 10** сама зависимость «глубина/фаза/возвраты/путь до выхода → поведение» берётся в работу; **числовые профили по этим признакам задаём отдельно для каждого типа** (Активный обходчик, Быстрый, Исследователь, Медленный).

## 3. Общие для всех типов (одна модель для всех агентов)

По этим признакам типы поведения **не различаются** (по данным проверки). Влияние на движение и остановки **одно и то же для всех** — задаём **общие правила**, без разбивки по типам.

| Гипотеза | Признаки | Что задавать |
|----------|----------|--------------|
| **1. Ширина прохода** | passage_width_mean | Одна зависимость для всех: уже проход → замедление, больше остановок; шире → быстрее, меньше остановок. |
| **2. Узкие места** | H2_min_width, H2_pct_narrow | Общие пороги и сила влияния узких участков для всех типов. |
| **3. Тупики и связность** | H3_mean_connectivity, H3_pct_low_connectivity | Общая зависимость: тупики → медленнее, меньше картин; проходные зоны → больше остановок. |
| **5. Коридоры vs залы** | H5_pct_corridor, H5_mean_aspect_ratio | Общая слабая зависимость: коридоры → чуть меньше просмотренных картин. |
| **6. Плотность экспонатов** | H6_mean_exhibit_density | Общая зависимость: выше плотность → выше скорость и больше экспонатов. |
| **9. Смена ширины** | H9_width_std, H9_width_range | Общая зависимость: больше смена → медленнее визит, меньше картин. |
| **11. Близость к лестницам** | H11_* (расстояние до лестниц, доля у лестниц) | Общая зависимость: ближе к лестницам → замедление, больше остановок. |
| **12. Количество поворотов** | H12_n_turns, H12_turns_per_* | Общая зависимость: больше поворотов по маршруту → дольше визит, больше остановок; больше поворотов в минуту → выше скорость. |
| **Изовист** | isovist_area_mean, isovist_area_* | Общая зависимость: больше видимая площадь → выше скорость, меньше остановок; больше видимость/разброс → меньше nb_items. |
| **Открытость/размер зон** | zone_area_mean, pct_small_zone, pct_large_zone | Общие зависимости (если используем) для всех типов. |

**Итог:** Для гипотез **1, 2, 3, 5, 6, 9, 11, 12** и изовиста — **берём в работу подтверждённые эффекты**, но **не делаем разных параметров по типам**: один и тот же закон «узко → медленнее», «видимо → быстрее» и т.д. для всех агентов.
